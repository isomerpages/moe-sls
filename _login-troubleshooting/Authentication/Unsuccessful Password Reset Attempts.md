<h1 id="unsuccessful-password-reset-attempts">Unsuccessful Password Reset Attempts</h1>
<h2 id="-incorrect-answers-to-security-questions-">Incorrect Answers to Security Questions</h2>
<hr>
<ol>
<li><p>If you answer any of the questions incorrectly, you will be brought to the Password Reset Unsuccessful page.</p>
<p><img alt="Unsuccessful Password Reset Attempts" style="width: 100%;" src="/images/4Troubleshooting/LT-UnsuccesfulPassword.png"></p>
</li><li>Click <strong>Retry</strong> to try again.</li>
<li>If you answer all the questions correctly, you will be brought to the Reset Password page.
</li><li>Enter a new password&nbsp;twice to confirm that you have entered it correctly.</li>
<li>Click <strong>Submit</strong>.</li>
<li>If your password has been successfully reset, you will be&nbsp;redirected&nbsp;brought&nbsp;to the login page.</li>
</ol>
