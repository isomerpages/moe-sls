<h1>About Self-Study Resources</h1>
<hr>
<p>The Self-Study List shows resources that you are currently attempting
or have previously attempted. You can access and review these resources
at your own pace.</p>
<p>When you attempt a Module from any of the following locations, it is automatically
added to the Self-Study List:</p>
<p><strong>Note:</strong>
</p>
<ul data-tight="true" class="tight">
<li>
<p>Groups older than four years are automatically archived and cannot be
accessed.</p>
</li>
<li>
<p>Self-Study attempts from archived Group Resources will not appear in the
Self-Study List.</p>
</li>
</ul>
<h2>Access Self-Study resources</h2>
<p>You can access Self-Study resources from the following locations.</p>
<h3>MOE Library</h3>
<p>The MOE Library contains resources created and curated by the Ministry
of Education (MOE). These resources support your learning across different
subjects and levels.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" src="/images/1Student/d_selfstudyresources2.png">
</div>
<p>To attempt a Module from the MOE Library:</p>
<ol data-tight="true" class="tight">
<li>
<p>Go to MOE Library.</p>
</li>
<li>
<p>Select a Module.</p>
</li>
<li>
<p>Click <strong>Attempt</strong>.</p>
</li>
</ol>
<p>Click <strong><a href="/student-user-guide/discover/attempt-self-study-resources/" rel="noopener noreferrer nofollow" target="_blank">here</a></strong> to
learn how to attempt MOE Library modules.</p>
<h2>Group Resources </h2>
<p>Group Resources are Modules that are uploaded by your teachers to the
Class Group. These modules are not assignments and do not have a due date.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" src="/images/1Student/d_selfstudyresources1.jpg">
</div>
<p>To find and sort resources in a Group:</p>
<ol data-tight="true" class="tight">
<li>
<p>Go to<strong> Groups</strong>.</p>
</li>
<li>
<p>Select a Group.</p>
</li>
<li>
<p>Click the <strong>Resources</strong> tab.</p>
</li>
<li>
<p>To filter resources:</p>
<ul data-tight="true" class="tight">
<li>
<p>Select a category from the dropdown list.</p>
</li>
<li>
<p>Click <strong>Apply</strong>.</p>
</li>
</ul>
</li>
<li>
<p>To sort resources:</p>
<ul data-tight="true" class="tight">
<li>
<p>Click <strong>Sort By</strong>.</p>
</li>
<li>
<p>Select one of the following:</p>
<ul data-tight="true" class="tight">
<li>
<p>Modified date</p>
</li>
<li>
<p>Date added</p>
</li>
<li>
<p>Title</p>
</li>
</ul>
</li>
</ul>
</li>
</ol>
<p>Click <strong><a href="/student-user-guide/organise/access-class-groups/" rel="noopener noreferrer nofollow" target="_blank">here</a></strong> to
learn how to Access Groups.</p>
<h2>My Drive</h2>
<p>My Drive contains Modules that you have copied from:</p>
<ul data-tight="true" class="tight">
<li>
<p>MOE Library</p>
</li>
<li>
<p>Class Group Resources</p>
</li>
<li>
<p>Assignments</p>
</li>
</ul>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" src="/images/1Student/d_selfstudyresources3.png">
</div>
<p>You can use these Modules for revision, including reattempting Assignments.</p>
<p>To search for Modules in My Drive:</p>
<ol data-tight="true" class="tight">
<li>
<p>Go to <strong>My Drive</strong>.</p>
</li>
<li>
<p>Use the search bar to find a Module</p>
</li>
</ol>
<p>Click <strong><a href="/student-user-guide/organise/search-in-my-drive/" rel="noopener noreferrer nofollow" target="_blank">here</a></strong> to
find out more about Searching in My Drive.</p>
