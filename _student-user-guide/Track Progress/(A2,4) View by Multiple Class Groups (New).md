<h1>(A2,4) View by Multiple Class Groups (New)</h1>
<hr>
<p>Students can select multiple class groups to view their learning progress
for assignments and self-study modules, Adaptive Learning System (ALS),
and Error Tracker.</p>
<ol data-tight="true" class="tight">
<li>
<p>From the homepage, open the left menu and click <strong>Learning Progress</strong>.</p>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="View by Multiple Class Groups" src="/images/1Student/T_View_Multiple.png">
</div>
</li>
<li>
<p>Select a subject to track learning progress.</p>
</li>
<li>
<p>Select the Class Group and Academic Year by clicking the dropdown arrow.</p>
</li>
<li>
<p>Next, select Assignment or Self-Study modules to view your learning progress.</p>
</li>
</ol>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="View by Multiple Class Groups" src="/images/1Student/T_View_Multiple1.png">
</div>
<p><u>Note</u>: You can view both MOE’s and your own school’s custom content
maps in Learning Progress.</p>
<p></p>
