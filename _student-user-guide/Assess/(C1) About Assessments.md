<h1>(C1) About Assessments</h1>
<hr>
<p>Assessments are Assignments with a special setting that teachers can use
for better control. In an Assessment, you can only access external links
if they have been whitelisted for use in the assessment's network.</p>
<h2>Logging in for E-Assessments</h2>
<hr>
<ol data-tight="true" class="tight">
<li>
<p>On the main login page, select your preferred login method.</p>
</li>
<li>
<p>If you had selected Login with SLS, select Login for E-Assessment.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/1Student/e_assessment_sls_login.png">
</div>
</li>
<li>
<p>Alternatively, you may Login with MIMS and select Login for E-Assessment.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/1Student/e_assessment_mims_login.png">
</div>
</li>
<li>
<p>Once you have logged in, you will see a list of Assessments that have
been assigned to you. If your teacher has set an Access Code for the Assessment,
you will be required to enter the Access Code (case-sensitive) in order
to access the Assessment.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/1Student/e_assessment_landing_page.png">
</div>
</li>
</ol>
<h2>Viewing Assessments</h2>
<hr>
<ol data-tight="true" class="tight">
<li>
<p>Assessments can only be accessed from this Assessment List.</p>
</li>
<li>
<p>You can view assessments directly by logging in for e-assessments or by
clicking on the Assessments button at the top right corner in the Assignment
List.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/1Student/E_Assessment_List.png">
</div>
</li>
<li>
<p>If your teacher has set an Access Code for the Assessment, you will be
required to enter the Access Code (case-sensitive) in order to access it.</p>
</li>
</ol>
<h2>Attempting Assessments</h2>
<hr>
<ol data-tight="true" class="tight">
<li>
<p>When you type a response in a Free-Response question, or when your audio
recording is being processed, the auto-save indicator will reflect 1 of
the 3 statuses: (1) Autosaving as draft, (2) Saved as draft, or (3) Autosaving
failed.</p>
<p></p>
<p><strong>If your internet connection goes offline:</strong>
</p>
<ul>
<li>
<p>Your online status next to your SLS profile will change to&nbsp;offline.
If you are typing a response in a Free-Response question, or if your audio
recording is being processed, the auto-save indicator will show <strong>Autosaving failed</strong>.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/1Student/E_Assessment_Offline.png">
</div>
</li>
<li>
<p>Please <strong><u>do not close</u></strong><u> </u>the browser page. Your
last saved response in SLS will be restored when connectivity resumes,
even if the page is refreshed.</p>
</li>
<li>
<p>Inform your teacher when this happens.</p>
</li>
</ul>
</li>
</ol>
