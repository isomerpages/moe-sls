<h2>Changing Text Alignment and Style</h2>
<hr>
<ol>
<li>
<p>In the Rich Text Editor, you are able to align your text left/center/right/justify,
change the font type and paragraphing style.</p>
</li>
<li>
<p>To adjust the text alignment, click</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/TextAlign.svg">
</div>
<p>and select an alignment type.</p>
</li>
<li>
<p>Click the <strong>dropdown arrow</strong> beside “English” to change the
font type.</p>
</li>
<li>
<p>To change the paragraphing style, click the <strong>dropdown arrow</strong> beside
“Paragraph”.</p>
<p><u>Note</u>: To ‘adjust’ the font size, you can select a different paragraphing
style (i.e. heading style).</p>
</li>
</ol>
