<h2>Submit a Lesson for Approval</h2>
<p>Content Officers (COs) and Content Approvers (CAs) can create and edit
MOE Library modules directly in “Manage Modules”</p>
<ol data-tight="true" class="tight">
<li>
<p>Go to Manage Modules under ‘Admin’.</p>
</li>
<li>
<p>Select your owner group to create your MOE Library module.</p>
