<h1>(B1,i) Generate CCPM Reports</h1>
<ol>
<li>
<p>Access to module and media utilisation data on SLS has been enhanced with
the <strong>SLS Modules Dashboard.</strong>
</p>
</li>
<li>
<p>Please use the <a href="" rel="noopener noreferrer nofollow" target="_blank">SLS Modules Dashboard</a> (on
WOG only) to download the module and media utilisation data.
<br>Available data:</p>
<ul data-tight="true" class="tight">
<li>
<p>Module utilisation data: from January 2024 onwards.</p>
</li>
<li>
<p>Media utilisation data: &nbsp;from August 2025 onwards.</p>
</li>
</ul>
</li>
<li>
<p>Click the<strong> eye icon&nbsp;</strong>to enter the dashboard.</p>
</li>
<li>
<p>On the top of the dashboard, select the <strong>MOE Library</strong> tab
(for the module utilisation report) or <strong>Media Utilisation</strong> tab
(for the media utilisation report).</p>
</li>
<li>
<p>In the filter bar, select your <strong>Owner Group </strong>and the <strong>Selected Months</strong> of
the data required. Note that if you select multiple months, numbers generated
will be the total for the selected months. For "number of unique students"
(MOE Library) or “unique number of times” (Media Utilisation) metrics,
which counts the number of unique users per month, this will be the total
for the selected months; users who accessed multiple times across the months
will be counted multiple times.</p>
</li>
<li>
<p>Click the <strong>download icon</strong> at the bottom and select <strong>Crosstab icon</strong>.
In the pop-up that appears, check that “MOE Library Full Set” (for the
module utilisation report) or “Media Utilisation Report” is selected, then
click <strong>Download</strong>.</p>
</li>
</ol>
