<h1 id="about-manage-users">About Manage Users</h1>
<p>SLS School Administrators are able to view the Names, SLS User ID, Role(s), Form Class, Index Number, Last Login Date, MIMS Status, SLS Account Status, Last Reset Date, Last Reset By, and Email for all teachers and students of the school. Most of these fields can be sorted in ascending/descending order. Click the <strong>Up/Down</strong> arrows in the table header to sort the listing.</p>
<ol>
<li>On the Manage Users page, search for a user in your school based on SLS user ID, Email, or Name.</li>
<li><p>To refine the search results, select the following filters from the respective dropdown list:</p>
<ul>
<li><strong>SLS Account Status</strong> - Activated, Activated (Locked), Not Activated, Not Activated (Locked), Inactive</li>
<li><p><strong>Form Class</strong> (for students only)</p>
<p><img style="width: 80%;" src="/images/5Admin/A-Filter1.png"></p>
</li>
</ul>
</li>
<li><p>Click <strong>Filter</strong> to select additional filters:</p>
<ul>
<li><strong>MIMS Status</strong> - Active, Inactive, Not Applicable</li>
<li><strong>School</strong></li>
<li><p><strong>Role</strong> - School Admin, Teacher, Student</p>
<p><img style="width: 100%;" src="/images/5Admin/A-SearchUsers.png"></p>
</li>
</ul>
</li>
<li><p>Click <strong>Apply</strong> to confirm your selection(s) or <strong>Reset</strong> to start again.</p>
</li>
</ol>
