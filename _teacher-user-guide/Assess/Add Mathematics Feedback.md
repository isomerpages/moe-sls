<h1>Add Mathematics Feedback</h1>

<p>Mathematics Feedback is an evaluation engine that offers step-by-step feedback to students on their input. </p>
<p><u>Important</u>:</p>
<ul>
<li>Mathematics Feedback can be added as Prepopulated Student Answer in a Free-Response Question. Students will be informed that Mathematics Feedback will provide feedback for the question.</li>
<li>File Submission and Suggested Answer Field are disabled when using Mathematics Feedback.</li>
</ul>
<h2 id="-create-a-mathematics-feedback-question-"><strong>Create a Mathematics Feedback question</strong></h2>
<hr>
<ol>
<li><p>Select <strong>Free-Response</strong> then <strong>Mathematics Feedback</strong> under Questions in the Component Bar. The Create New Question subpage will open.</p>
<ul>
<li><p>Alternatively, you can choose Default Free-Response then click the checkbox next to <strong>Use</strong> and select <strong>Mathematics Feedback</strong>. In the Prepopulated Student Answer, click the <strong>Create a Mathematics question</strong> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/algebrakit.svg"> to open the Create New Question subpage.</p>
</li>
</ul>
</li>
<img style="width: 100%;" src="/images/2Teacher/AS-AddMathematicsFeedback01.png">
<li><p>In the Create New Question subpage, check the box whether you would like to show hints and feedback while students attempt. </p></li>
<img style="width: 100%;" src="/images/2Teacher/AS-AddMathematicsFeedback02.png">
<li><p>To learn more about creating Mathematics Feedback questions and the different question types that are supported, click <a target="_blank" href="https://help.algebrakit.com">here</a>. </p>
</li>
<li>When you are done, click <strong>Save</strong>.</li>
</ol>
