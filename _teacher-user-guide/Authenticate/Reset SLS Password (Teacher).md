<h3>Reset SLS Password (Teacher)</h3>
<ol data-tight="true" class="tight">
<li>
<p>From the main login page, click <strong>Login With SLS</strong>.</p>
</li>
<li>
<p>Click <strong>Forgot Password</strong>.</p>
</li>
<li>
<p>Enter your SLS Username.</p>
</li>
<li>
<p>Click <strong>Submit</strong>.</p>
</li>
<li>
<p>An email with the password reset link will be sent to your email address.</p>
</li>
<li>
<p>Click the reset password link in the email.</p>
<ul data-tight="true" class="tight">
<li>
<p>The link is valid for 10 minutes.</p>
</li>
<li>
<p>If the link has expired, you can request another by repeating the steps
above.</p>
</li>
</ul>
</li>
<li>
<p>On the <strong>Reset Password</strong> page, enter a new password.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/4Troubleshooting/Reset_Password.png">
</div>
<ul data-tight="true" class="tight">
<li>
<p>You will need to enter the new password twice to confirm that you have
entered it correctly.</p>
</li>
</ul>
</li>
<li>
<p>Click <strong>Submit</strong>.</p>
</li>
<li>
<p>If your password reset is successful, you will be brought to the login
page.</p>
</li>
<li>
<p>Click <a href="/login-troubleshooting/authentication/sls-password-reset-email/" rel="noopener noreferrer nofollow" target="_blank">here</a> if
you did not receive your password reset email</p>
</li>
</ol>
<h3>Unlock Account</h3>
<hr>
<p>For security reasons, your account will be locked if you try to log in
with an incorrect password too many times. Please approach your school's
SLS administrator for help with unlocking your account.</p>
