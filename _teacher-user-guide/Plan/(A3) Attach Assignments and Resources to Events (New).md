<h1>(A3) Attach Assignments and Resources to Events (New)</h1>
<p>You can attach Assignments and Resources from My Drive, Library and Starred Resources to an Event in your Planner. The total number of Assignments and Resources attached is indicated below the Event title.</p>
<h3>Attaching Assignments to an Event</h3>
<hr>
<ol>
    <li>
        <p>Enter the Planner page or the Planner tab from a selected Class Group.</p>
    </li>
    <li>
        <p>On the Event card, click the <strong>Plus</strong> icon and select <strong>Attach Assignments</strong>.</p>
    </li>
    <li>
        <p>Alternatively, click the <strong>Event</strong> card to open the Event Details subpage.</p>
    </li>
    <li>
        <p>Click <strong><strong><strong><strong><strong><strong>Attach</strong></strong></strong></strong></strong></strong> to attach an assignment, or click <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>Quick Assign</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong> to select between assigning a new Activity [hyperlink <a href="https://www.notion.so/Add-New-Activities-Merge-with-Add-and-Edit-Activities-Sections-80583d3cf8934af6994d3655383d5c52?pvs=21">Add New Activities</a>] or Quiz [hyperlink <a href="https://www.notion.so/1b-Add-and-Edit-Quizzes-New-b868eb6f0d6a4c378fd1bc9f4d88014a?pvs=21">Add and Edit Quizzes</a>].</p>
        <p>Note: Quick Assign is only applicable for Class Group Events.</p>
    </li>
    <li>
        <p>You can attach assignments from the Attach Assignments subpage. You can search for Assignments based on title, Assignment Status, and/or Class Group.</p>
    </li>
    <li>
        <p>Click <strong><strong><strong><strong><strong><strong>Filter</strong></strong></strong></strong></strong></strong> to filter your search based on Marking Status. Click <strong><strong><strong><strong><strong>Apply</strong></strong></strong></strong></strong> to proceed. [insert screenshot]</p>
    </li>
    <li>
        <p>From the Attach Assignment subpage, you can also view the details, manage, move the Assignment to a different Class Group or to a Class Group category. To monitor the Assignment, click the <strong>Monitoring</strong> icon on the Assignment card. [insert screenshot showing expanded overflow]</p>
    </li>
    <li>
        <p>To select an Assignment to attach, click the <strong>Assignment card</strong>. This will bring your back to the Event Details subpage, with the selected Assignment attached. [insert screenshot].</p>
    </li>
</ol>
<h3>Attaching Resources to a Personal Event</h3>
<hr>
<ol>
    <li>Enter the Planner page or the Planner tab from a selected Class Group.</li>
    <li>On the Personal Event card, click the <strong><strong><strong><strong>Plus</strong></strong></strong></strong> icon and select <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From My Drive</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong>, <strong><strong><strong><strong><strong><strong>From Library</strong></strong></strong></strong></strong></strong>, or <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From Starred Resources</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong> under Attach Resources. [insert screenshot]</li>
    <li>Alternatively, click the <strong>Event</strong> card to open the Event Details subpage. Click <strong><strong><strong><strong><strong><strong><strong><strong><strong>Attach</strong></strong></strong></strong></strong></strong></strong></strong></strong> and select a resource location - <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From My Drive</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong>, <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From Library</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong>, <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From Starred Resources</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong>.<ol>
            <li>From My Drive: select between My Lessons and Shared with Me by clicking <strong><strong><strong><strong>Open</strong></strong></strong></strong> below the respective folder title. Select a Lesson by clicking the radio button, followed by <strong><strong><strong><strong><strong><strong>Choose</strong></strong></strong></strong></strong></strong>.</li>
            <li>From Library: search for a Lesson [hyperlink search for resources] from MOE Library or Community Gallery. Select a <strong>Lesson card</strong> to add that particular lesson as a resource. Click the <strong>View Details</strong> icon to view the lesson details. You may select <strong><strong><strong><strong><strong><strong>Choose</strong></strong></strong></strong></strong></strong> to add the Lesson as a resource, or <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>Back to Attach Resources</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong> to select a different Lesson.</li>
            <li>From Starred Resources: Select a <strong>Lesson card</strong> to add that particular lesson as a resource.</li>
        </ol>
    </li>
    <li>After selecting a resource, you will be brought back to the Event Details subpage, with the selected resource attached [insert screenshot]</li>
</ol>
<h3>Attaching Resources to a Class Group Event</h3>
<hr>
<ol>
    <li>Enter the Planner page or the Planner tab from a selected Class Group.</li>
    <li>On the Class Group Event card, click the <strong>Plus</strong> icon and select <strong>Attach Resources</strong>.</li>
    <li>Alternatively, click the <strong>Event</strong> card to open the Event Details subpage. Any Class Group Resources will be listed here. You can search for Assignments based on title, Class Group Resource Status, and/or Category. [insert screenshot]</li>
    <li>From the Attach Class Group Resources subpage, you can also view the details, make a copy, move the Lesson to a different Class Group or to a Class Group category. [insert screenshot showing expanded overflow]</li>
    <li>Select a <strong>Lesson card</strong> to attach as a Class Group Resource. This will bring your back to the Event Details subpage, with the selected resource attached. [insert screenshot].</li>
    <li>To attach a resource from other locations, click <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>Add Resource</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong> and select a resource location - <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From My Drive</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong>, <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From Library</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong>, <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>From Starred Resources</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong>.<ol>
            <li>From My Drive: select between My Lessons and Shared with Me by clicking <strong><strong><strong><strong>Open</strong></strong></strong></strong> below the respective folder title. Select a Lesson by clicking the radio button, followed by <strong><strong><strong><strong><strong><strong>Choose</strong></strong></strong></strong></strong></strong>.</li>
            <li>From Library: search for a Lesson [hyperlink search for resources] from MOE Library or Community Gallery. Select a <strong>Lesson card</strong> to add that particular lesson as a resource. Click the <strong>View Details</strong> icon to view the lesson details. You may select <strong><strong><strong><strong><strong><strong>Choose</strong></strong></strong></strong></strong></strong> to add the Lesson as a resource, or <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>Back to Add from Library</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong> to select a different Lesson.</li>
            <li>From Starred Resources: Select a <strong>Lesson card</strong> to add that particular lesson as a resource.</li>
        </ol>
    </li>
    <li>Resource that is selected from My Drive, MOE Library, Community Gallery, and Starred Resources will redirect you to the Resource Title subpage. Enter the Lesson Title, select a Category (optional), adjust the Permission Settings (optional), and click <strong><strong><strong><strong><strong><strong>Create</strong></strong></strong></strong></strong></strong>. [insert screenshot]</li>
    <li>Click <strong><strong>OK</strong></strong> to the confirmation prompt. Click <strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong><strong>Back to Planner</strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong></strong> to return to the Planner page or Planner tab within your selected Class Group.</li>
</ol>
<h3>Detaching an Assignment or Resource from an Event</h3>
<hr>
<ol>
    <li>
        <p>Click the <strong>Event card</strong> to enter the Event Details subpage.</p>
    </li>
    <li>
        <p>To detach an Assignment, click the <strong>Overflow</strong> (<strong>3</strong>-<strong>dot</strong> icon) and select <strong>Detach Task</strong>.</p>
    </li>
    <li>
        <p>To detach a Resource, click the <strong>Detach</strong> icon on the Resource card.</p>
        <p>[insert screenshot to show both #2 and 3]</p>
    </li>
</ol>
