<h1 id="assign-moe-library-modules">(A1,iii) Assign MOE Library Modules (Enhanced)</h1><hr>
<h2 id="assign-moe-module-as-assignment">Assign MOE Module as Assignment</h2>
<hr>
<ol>
<li>From the homepage, open the left menu and click <strong>Resources</strong>, followed by <strong>MOE Library</strong>.</li>
<li>Click the MOE Library <strong>Lesson card</strong> that you would like to view.</li>
<li>On the Lesson View page, click the <strong>Assign</strong> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/Assign.svg"> on the header.</li>
<li>You may wish to rename the Assignment Title and edit the Permission Settings. The following Lesson Permissions are available:
<ol style="list-style-type: lower-alpha;">
<li><strong>Allow teachers to copy/assign/publish and students to copy this Lesson</strong></li>
<li><strong>Allow teachers and students to view Print-Friendly worksheet</strong></li>
<li><strong>Allow students to reattempt Lesson/Course as Self-Study</strong></li>
</ol>
</li>
<li>Click <strong>Next</strong> to continue. This will lead you to the <a target="_blank" href="/teacher-user-guide/assign/manage-assignments/">Assignment Summary</a> page where you can review the draft assignments and their settings before clicking <strong>Assign</strong> to confirm.</li>
<li>Click on the right menu and select <strong>Access via QR Code</strong> to generate a shortened URL and QR Code with SLS branding for your students to access this module. After this, the Student Attempt URL will automatically change to the shortened URL.</li>
<p><img alt="Assign MOE Library Modules" style="width: 80%;" src="/images/2Teacher/ASS_QRcode_3B.png"></p>
</ol>
<h2 id="add-moe-module-as-class-group-resource">Add MOE Module as Class Group Resource</h2>
<hr>
<p><strong>Important</strong>: If lesson permissions to allow teachers to copy/assign/publish and students to copy this lesson are not selected, it will not be possible to add the lesson as a Class Group Resource.</p>
<ol>
<li><p>Click the MOE Library <strong>Lesson card</strong> that you would like to view.</p>
</li>
<li><p>Click <strong>Overflow</strong> (<strong>3 dots</strong> icon) on the header to open the right menu and click <strong>Add to Class Group Resources</strong>.</p>
<p><img alt="Assign MOE Library Modules" style="width: 50%;" src="/images/2Teacher/AS-AssignMOE1.png"></p>
</li>
<li><p>Select the Class Group(s), followed by <strong>Add</strong>.</p>
</li>
<li><p>You may wish to rename the Resource Title, and edit the Permission Settings:</p>
<ol style="list-style-type: lower-alpha;">
<li>Lesson or Course Permissions:<ol style="list-style-type: lower-roman;">
<li><strong>Allow teachers to copy/assign/publish and students to copy this Lesson</strong></li>
<li><strong>Allow teachers and students to view Print-Friendly worksheet</strong></li>
<li><strong>Allow students to reattempt Lesson/Course as Self-Study</strong></li>
</ol>
</li>
<li><p>Visibility</p>
<ol style="list-style-type: lower-roman;">
<li><strong>Visible to students in Class Group</strong></li>
<li><strong>Notify Student(s)</strong> - this checkbox will be active if the resource is set as visible to students in Class Group.</li>
</ol>
<p><img alt="Assign MOE Library Modules" style="width: 100%;" src="/images/2Teacher/AS-AssignMOE2.png"></p>
</li>
</ol>
</li>
<li><p>Click <strong>Create</strong> to add the lesson as a Class Group Resource.</p>
</li>
</ol>
