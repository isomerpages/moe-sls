<h1>(A1,ii) Assign MOE Library Modules (Enhanced)</h1>
<hr>
<h2>Assign MOE Module as Assignment</h2>
<hr>
<ol data-tight="true" class="tight">
<li>
<p>From the homepage, open the left menu and click <strong>Resources</strong>,
followed by <strong>MOE Library</strong>.</p>
</li>
<li>
<p>Click the MOE Library <strong>Lesson card</strong> that you would like to
view.</p>
</li>
<li>
<p>On the Lesson View page, click the <strong>Assign</strong> icon</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/Assign.svg">
</div>
<p>on the header.</p>
</li>
<li>
<p>You may wish to rename the Assignment Title and edit the Permission Settings.
The following Lesson Permissions are available:</p>
<ol data-tight="true" class="tight">
<li>
<p><strong>Allow teachers to copy/assign/publish and students to copy this Lesson</strong>
</p>
</li>
<li>
<p><strong>Allow teachers and students to view Print-Friendly worksheet</strong>
</p>
</li>
<li>
<p><strong>Allow students to reattempt Lesson/Course as Self-Study</strong>
</p>
</li>
</ol>
</li>
<li>
<p>Click <strong>Next</strong> to continue. This will lead you to the <a href="/teacher-user-guide/assign/manage-assignments/" rel="noopener noreferrer nofollow" target="_blank">Assignment Summary</a> page
where you can review the draft assignments and their settings before clicking <strong>Assign</strong> to
confirm.</p>
</li>
<li>
<p>Click on the right menu and select <strong>Access via QR Code</strong> to
generate a shortened URL and QR Code with SLS branding for your students
to access this module. After this, the Student Attempt URL will automatically
change to the shortened URL.</p>
</li>
</ol>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/ASS_QRcode_3B.png">
</div>
<h2>Add MOE Module as Class Group Resource</h2>
<hr>
<p><strong>Important</strong>: If lesson permissions to allow teachers to
copy/assign/publish and students to copy this lesson are not selected,
it will not be possible to add the lesson as a Class Group Resource.</p>
<ol>
<li>
<p>Click the MOE Library <strong>Lesson card</strong> that you would like to
view.</p>
</li>
<li>
<p>Click <strong>Overflow</strong> (<strong>3 dots</strong> icon) on the header
to open the right menu and click <strong>Add to Class Group Resources</strong>.</p>
<div class="isomer-image-wrapper">
<img style="width: 50%;" height="auto" width="100%" alt="Assign MOE Library Modules" src="/images/2Teacher/AS-AssignMOE1.png">
</div>
</li>
<li>
<p>Select the Class Group(s), followed by <strong>Add</strong>.</p>
</li>
<li>
<p>You may wish to rename the Resource Title, and edit the Permission Settings:</p>
<ol>
<li>
<p>Lesson or Course Permissions:</p>
<ol data-tight="true" class="tight">
<li>
<p><strong>Allow teachers to copy/assign/publish and students to copy this Lesson</strong>
</p>
</li>
<li>
<p><strong>Allow teachers and students to view Print-Friendly worksheet</strong>
</p>
</li>
<li>
<p><strong>Allow students to reattempt Lesson/Course as Self-Study</strong>
</p>
</li>
</ol>
</li>
<li>
<p>Visibility</p>
<ol data-tight="true" class="tight">
<li>
<p><strong>Visible to students in Class Group</strong>
</p>
</li>
<li>
<p><strong>Notify Student(s)</strong> - this checkbox will be active if the
resource is set as visible to students in Class Group.</p>
</li>
</ol>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" alt="Assign MOE Library Modules" src="/images/2Teacher/AS-AssignMOE2.png">
</div>
</li>
</ol>
</li>
<li>
<p>Click <strong>Create</strong> to add the lesson as a Class Group Resource.</p>
</li>
</ol>
<p>Note: As MOE Library modules may be updated periodically, some modules
may be archived. Students can still access archived module versions which
they have previously attempted or previously starred. Teachers can also
access archived module versions they have previously starred. They will
see a banner at the top of archived module versions that includes a link
to the most updated version</p>
