<h1>(A2,4) View by Multiple Class Groups (New)</h1>
<p>Teachers can select multiple class groups to view students’ learning progress for assignments and self-study modules, Adaptive Learning System (ALS), and Error Tracker, across multiple class groups. Teachers can also generate reports for multiple class groups for assignments and self-study modules, and ALS.</p>
<ol>
    <li>
        <p>From the homepage, open the left menu and click <strong>Learning Progress</strong>. Select a subject to track learning progress.</p>
        <p><img alt="View by Multiple Class Groups" style="width: 100%;" src="/images/2Teacher/T_View_Multiple.png"></p>
    </li>
    <li>You can view both MOE’s and your school’s custom content maps in Learning Progress. Select the Class Group and Academic Year by clicking the dropdown arrow. </li>
<li>Next, select Assignment or Self-Study to view students’ learning progress. Select All to view learning progress across both Assignments and Self-Study modules.</li>
    
    <li>
        <p>You may also click <strong>Download Report</strong> to generate learning reports for multiple Class Groups.</p>
    </li>
<p><img alt="View by Multiple Class Groups" style="width: 100%;" src="/images/2Teacher/T_View_Multiple1.png"></p>
</ol>
