<h1>(E1) Access My Drive</h1>
<hr>
<p><strong>My Drive</strong> can be accessed from the homepage by clicking
on the Folder icon on the left menu.</p>
<h2>About My Drive</h2>
<p>Once you are in <strong>My Drive</strong>, you can access the following
sections easily:</p>
<ul data-tight="true" class="tight">
<li>
<p><strong>My Modules</strong>
</p>
</li>
<li>
<p><strong>Shared with Me</strong>
</p>
</li>
<li>
<p><strong>Trash</strong>
</p>
</li>
</ul>
