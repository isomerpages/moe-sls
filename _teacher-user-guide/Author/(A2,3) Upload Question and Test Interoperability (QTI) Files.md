<h1>(A2,3) Upload Question and Test Interoperability (QTI 2.1) Files</h1>
<hr>
<h2>Digitise Past Year Papers for Practice</h2>
<p>You can digitise your school's past year exam questions from within SLS
for your students' revision.</p>
<ol data-tight="true" class="tight">
<li>
<p>Upload your past year paper (in Word or PDF format) to a QTI converter
tool.</p>
</li>
<li>
<p>Download the converted file as a QTI 2.1 format.</p>
</li>
<li>
<p>Navigate to the Module editor to upload the QTI 2.1 file.</p>
</li>
<li>
<p>Your questions will be automatically populated in the Module.</p>
</li>
<li>
<p>When assigning or managing the Module, <a href="/teacher-user-guide/assign/allow-students-to-copy-assignments/" rel="noopener noreferrer nofollow" target="_blank">allow students to copy the Assignment</a> to
their My Drive folder to enable self-study.</p>
</li>
</ol>
<h2>Uploading a QTI 2.1 ZIP file in SLS</h2>
<hr>
<ol>
<li>
<p>Ensure your QTI file is exported as QTI 2.1 version.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" src="/images/2Teacher/AU-AddQTI1.png">
</div>
</li>
<li>
<p>In the Module Editor page, hover over Question in the Component Bar.</p>
<div class="isomer-image-wrapper">
<img style="width: 50%;" height="auto" width="100%" src="/images/2Teacher/AU-AddQTI2.png">
</div>
</li>
<li>
<p>Under Add From Existing, select Questions from file (QTI).</p>
</li>
<li>
<p>Upload a QTI zip file into the Import Question subpage and click <strong>Upload</strong>.
You can also import images into options using QTI.</p>
<p><u>Note</u>:</p>
<ul data-tight="true" class="tight">
<li>
<p>A sample file can be found <a href="https://go.gov.sg/qtisample" rel="noopener noreferrer nofollow" target="_blank">here</a>. (Q1-15 will be uploaded successfully
while Q16 will be unsuccessful as it is meant to test for errors.)</p>
</li>
</ul>
</li>
<li>
<p>If the file is valid, it will be uploaded and scanned for viruses.</p>
<p><u>Note</u>: The virus scan will take longer if the file size is large.</p>
</li>
<li>
<p>When the file is successfully uploaded, the QTI questions will be added
to the Module. Each question in your zip folder will appear as a standalone
question within the Activity.</p>
</li>
<li>
<p>Should there be an error message indicating “processing partially unsuccessful”,
you may have to manually make some edits to the question.</p>
</li>
</ol>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" src="/images/2Teacher/AU-AddQTI3.png">
</div>
<h2>Considerations when creating a QTI File</h2>
<hr>
<p>When creating a QTI File for SLS, the maximum number of questions allowed
in SLS is 100 questions per file. Some guidelines to uploading a QTI file
include:</p>
<ol data-tight="true" class="tight">
<li>
<p>Only QTI 2.1 files are accepted.</p>
</li>
<li>
<p>Question requirements:</p>
<ol data-tight="true" class="tight">
<li>
<p>Questions stem/images</p>
</li>
<li>
<p>Answer options (if applicable)</p>
</li>
</ol>
</li>
<li>
<p>File Upload:</p>
<ol data-tight="true" class="tight">
<li>
<p>A recognised QTI 2.1 zipped file from any QTI conversion tool (e.g.
<a href="https://iwant2study.moe.edu.sg/lookangejss/QTIlowJunHua/docx_to_qti.html" rel="noopener nofollow" target="_blank">MOE DOCX to QTI Converter</a>or <a href="https://getmarked.ai/" rel="noopener noreferrer nofollow" target="_blank">getmarked.ai</a> ) with no more than 100 questions.</p>
</li>
</ol>
</li>
</ol>
<p><u>Note</u>:</p>
<ol data-tight="true" class="tight">
<li>
<p>Identifying the correct answers to Multiple-Choice Questions with single
or multiple responses can be done within SLS after the questions have been
uploaded.</p>
</li>
<li>
<p>You may add Hints, <a href="/teacher-user-guide/assess/add-teacher-comments/" rel="noopener noreferrer nofollow" target="_blank">Teacher Comments</a> and
<a href="/teacher-user-guide/author/add-question-tags/" rel="noopener noreferrer nofollow" target="_blank">Question Tags</a>to questions within SLS. Tagged questions will be reflected
in the <a href="/teacher-user-guide/track-progress/access-learning-progress/" rel="noopener noreferrer nofollow" target="_blank">Learning Progress</a> when
they have been attempted by students.</p>
</li>
</ol>
<h2>FAQs</h2>
<hr>
<ol>
<li>
<p><strong>Why does the QTI file fail to be accepted by SLS after uploading?</strong>
</p>
<p>Ensure the version of your QTI file is QTI 2.1.</p>
</li>
<li>
<p><strong>Where can I read more about QTI?</strong>
</p>
<p>You can read about it <a href="https://digitaliser.getmarked.ai/blog/complete-guide-to-qti/" rel="noopener noreferrer nofollow" target="_blank">here</a>.</p>
</li>
<li>
<p><strong>I tried downloading the zip file on my Safari browser but the file was unzipped automatically.</strong>
</p>
<p>Go to your Safari Preference and uncheck 'Open “safe” files after downloading'.
This should allow you to download the zip file without it being automatically
unzipped.</p>
</li>
</ol>
