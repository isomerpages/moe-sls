<h1>Use Authoring Copilot to create New Activities &amp; Components</h1>
<hr>
<p><u>Important:</u>
</p>
<ul data-tight="true" class="tight">
<li>
<p>As Authoring Copilot uses generative AI, its response is probabilistic
in nature. Thus, it may at times generate a different number of Sections,
Activities and Components than expected.</p>
</li>
<li>
<p>Check that the content generated by Authoring Copilot is accurate and
appropriate.</p>
</li>
</ul>
<p></p>
<p>To access Authoring Copilot to create components in an Activity, in the <strong>Create/Edit </strong>subpage,
click the <strong>Authoring Copilot icon</strong> on the Activity card.</p>
