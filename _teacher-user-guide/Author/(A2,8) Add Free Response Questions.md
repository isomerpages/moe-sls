<h1>(A2,8) Add Free-Response Questions</h1>
<hr>
<h2>Add Free-Response Question (FRQ)</h2>
<hr>
<ol>
<li>
<p>In the Module Editor page, hover over Question in the Component Bar.</p>
</li>
<li>
<p>Select <strong>Free-Response</strong>.</p>
</li>
<li>
<p>Below are the types of FRQ options you can choose from:</p>
<ul>
<li>
<p><strong>Default</strong>: Students will type their response in the answer
field.</p>
</li>
<li>
<p><strong>File Submission</strong>: Allow file submission checkbox will
be checked. If checked, students have the option of uploading a file when
they submit their responses. Students can upload up to 10 files. <a href="/teacher-user-guide/author/file-size-limits/" rel="noopener noreferrer nofollow" target="_blank">File limits</a> are
based on the file type.</p>
</li>
<li>
<p><strong>Google Response</strong>: You can upload a file from Google Drive
for students to type their response on.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: You need to <a href="/teacher-user-guide/customise/manage-linked-account/" rel="noopener noreferrer nofollow" target="_blank">link your iCON 2.0 / Google Account to SLS.</a>
</p>
</li>
</ul>
</li>
<li>
<p><strong>Annotated Feedback Assistant</strong>: It provides targeted in-line
feedback on both content and language through annotation cards, using suggested
answers, rubrics, and error tags to help your students refine their responses
before submitting.</p>
</li>
<li>
<p><strong>Feedback Assistant - Mathematics</strong>: The question set using
FA-Math will appear in the Pre-populated Student Answer Field. You can
edit the question by clicking the <strong>Pencil</strong> icon or delete
by clicking the <strong>Trash</strong> icon. Mathematics Feedback provides
feedback (and Marks if checked) to students after they submit their response.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: File Submission and Suggested Answer Field is disabled when
using Mathematics Feedback.</p>
</li>
</ul>
</li>
<li>
<p><strong>Short Answer Feedback Assistant</strong>: This tool provides suggested
grades and auto-generated content-related feedback for close-ended, short
answer questions for any subject and level.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: Short Answer Feedback Assistant is only available in Teacher-Marked
Quiz.</p>
</li>
</ul>
</li>
<li>
<p><strong>Interactive Response:</strong> This question is created by uploading
HTML5 ZIP files. This tool will automatically return marks from a HTML5
interactive to students after their attempt. Click <a href="/teacher-user-guide/author/html5-content-development/" rel="noopener noreferrer nofollow" target="_blank">here</a> to
learn how to create HTML5 interactive content.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: Interactive Response can be used in standalone FRQ and other
types of quizzes.</p>
</li>
</ul>
</li>
</ul>
</li>
<li>
<p>In the Question Body field, enter the question.</p>
</li>
<li>
<p>Click outside the Component to <strong>Save</strong>.</p>
</li>
</ol>
<h2>Edit Free-Response Question Details</h2>
<hr>
<p>Fill in the optional details of the Free-Response question:</p>
<ul>
<li>
<p>Recommended Time - You can set the recommended time for the student to
complete the</p>
</li>
<li>
<p>Response Size - You can adjust the response size (Short, Medium or Long)
using dropdown menu to choose the response size best suited for the question.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/AU_AddFRQ2.png">
</div>
</li>
<li>
<p>Pre-populated Student Answer - This allows you to insert an answer stem
to scaffold students' responses, or to provide an image for students to
work on. When students answer the question, they will be able to directly
edit this answer stem or image. Students can edit the images provided by
clicking the <strong>Edit Drawing</strong> icon and drawing on it.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: You can also use Google Response to Pre-populated Student
Answer.</p>
</li>
</ul>
</li>
<li>
<p>Add Rubrics - This allows students to view the rubrics while they attempt
the question.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: Show and use rubric marks checkbox - If checked, marks will
be defined by the Rubrics.</p>
</li>
</ul>
</li>
<li>
<p>Suggested Answer - The suggested answer to the question. This is shown
to students after they have submitted their answers.</p>
</li>
<li>
<p>Feedback - The explanation to the question's answer(s). This is shown
to students after they have submitted their answers.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: From March 2025 Update, you can also edit certain fields,
such as the Suggested Answer and Feedback fields, after the module has
been assigned and students have started attempting the question.</p>
</li>
</ul>
</li>
<li>
<p>Marks - You can set the number of marks. The default is one mark.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/author/edit-detail-cards/" rel="noopener noreferrer nofollow" target="_blank">here</a> for
editing the Question Component Details card.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/assess/edit-quizzes/" rel="noopener noreferrer nofollow" target="_blank">here</a> for
adding questions in a Quiz.</p>
</li>
</ul>
<p><u>Note</u>:</p>
<ul>
<li>
<p>From R4 (Jun 2026), the auto-save indicator will reflect 1 of the 3 statuses
when students are typing their responses: (1) Autosaving as draft, (2)
Saved as draft, or (3) Autosaving failed. Students may also manually click
on <strong>Save as Draft</strong>.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/AU_AddFRQ5.png">
</div>
</li>
<li>
<p>If internet connection is unstable, please inform students <strong>not to close</strong> the
browser page. Their last saved response in SLS will be restored when connectivity
resumes, even if the page is refreshed.</p>
</li>
</ul>
