<h1 id="add-existing-from-library-starred-resources">(A1,3) Add Existing From Library &amp; Starred Resources (Enhanced)</h1><hr>
<h2 id="viewing-existing-modules-from-library-starred-resources">Viewing Existing Modules From Library/Starred Resources</h2>
<hr>
<ol>
<li>When Adding from Library, you can search for Modules using the Search Bar which contains the following filter:
<ol style="list-style-type: lower-alpha;">
<li>You can add an existing Activity or Quiz from Library (MOE Library, Community Gallery, Class Group and Community Resources, and My Drive) or from Starred Resources.</li>
</ol>
<ul>
<li>Subject</li>
<li>Level</li>
</ul>
<p><u>Note</u>: Click <strong>Guided Search</strong> to search for specific words/phrases.</p>
</li>
<li>Click <strong>Apply</strong> to apply your filters.</li>
<li>You may use Filters for more targeted search results. To apply filters, click <strong>Filter</strong>. You can apply the following filters:
<ul>
<li>Location</li>
<li>Subject</li>
<li>Level</li>
<li>Add <a target="_blank" href="/teacher-user-guide/discover/search-for-resources/">Topic Filters</a> to select Content Map, and Topics and Sub-topics</li>
</ul>
</li>
<li>Click <strong>Apply Filters</strong> or <strong>Reset</strong>.</li>
<li>By default, the Search Results are sorted according to Relevance. You can choose to sort the results by recency by selecting Latest from the Sort By dropdown list.</li>
<p><img alt="Add Existing From Library &amp; Starred Resources" style="width: 100%;" src="/images/2Teacher/AU-AddExisting1.png"></p>
<li>Click the <strong>View Details</strong> icon on the Module card to view the following on the Module Details subpage:
<ul>
<li>Module title, cover and description (if any)</li>
<li>View Teacher Notes toggle - You can toggle on/off to view Teacher Notes (if any)</li>
<li>All Activities/Sections in the Module - Click <strong>Show Activities</strong> to see all Activities in a Section. The number of Activities in a Section is shown in parenthesis ( ).</li>
<li>View in new tab - Click the <strong>Expand</strong> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/external-link.svg"> to view a Module/Activity/Section in a new tab.</li>
</ul>
<p><u>Note</u>: This will not create a new Module in your Drive and you will be unable to edit the Module/Activity/Section.</p>
<ul>
<li>Other Information - Location, Subject and Level, Created by, Last Edited by, View Count and Keyword Tags</li>
</ul>
</li>
</ol>
<h2 id="choosing-an-existing-activity-section-from-library-starred-resources">Choosing an Existing Activity/Section From Library/Starred Resources</h2>
<hr>
<ol>
<li>You may search for modules from Library or Starred Resources.</li>
<li>Once you have decided on the Activities/Sections to add, you can tick the box beside the Activities/Sections.
<p><u>Note</u>:</p>
<ul>
<li>If you select a Section, all the Activities in that Section will be added.</li>
<li>Selection has to be of the same level (if you select a Section, you can only select another Section. If you select Activities, you can only select other Activities. I.e. You cannot add a Section with Activities from another Section).</li>
</ul>
</li>
<li>To clear the selected list of Activities/Sections, click <strong>Clear</strong>.</li>
<li>Click <strong>Choose</strong> to add the Activities/Sections into your Module.</li>
<p><img alt="Add Existing From Library &amp; Starred Resources" style="width: 100%;" src="/images/2Teacher/AU-AddExisting2.png"></p>
<li>The chosen Activities will inherit the Activity Details from the Module in Library/Starred Resources. To view and edit the Activity Notes and Active Learning Process in the <strong>Activity Details</strong> card, click the <strong>Settings</strong> icon on the <strong>Activity Details</strong> card. These fields are optional and will not be visible to students.</li>
</ol>
