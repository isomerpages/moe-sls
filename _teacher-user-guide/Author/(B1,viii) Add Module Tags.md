<h1>(B1,viii) Add Module Tags</h1>
<hr>
<p><strong>Important</strong>: Module Tags will be automatically added to
newly created sections.</p>
<p>Modules can be tagged to Subject, Level, Topics, and Sub-topics in the
MOE Content Map to improve searchability in the MOE Library, Community
Gallery, and My Drive.</p>
<p>If Modules are submitted to the Community Gallery, tags will allow teachers
to find the Module using filters and relevant search terms.</p>
<h2>Add Module Tags</h2>
<hr>
<ol>
<li>
<p>Click the&nbsp;<strong>Pencil</strong> icon</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/Pencil.svg">
</div>
<p>&nbsp;found at the top right of the header to enter the Module Editor
page.</p>
</li>
<li>
<p>Click on the Module cover page to edit the module details.</p>
</li>
<li>
<p>Click the dropdown arrow next to <strong>Module Tags</strong>.</p>
<div class="isomer-image-wrapper">
<img style="width: 70%;" height="auto" width="100%" alt="" src="/images/Module_Tag__1_.png">
</div>
</li>
<li>
<p>You can choose to select topics from the teaching subjects assigned to
you&nbsp;<strong>(Assigned Subjects)</strong>, subjects you are following <strong>(Followed Subjects),</strong>&nbsp;or
from any subject in the MOE Library&nbsp;<strong>(Other Subjects).</strong>
</p>
</li>
<li>
<p>Select the <strong>Subject</strong> from the dropdown list.</p>
</li>
<li>
<p>Select the <strong>Level</strong> from the dropdown list. This list is based
on the Subject chosen.</p>
</li>
<li>
<p>Select the <strong>Content Map</strong> and <strong>Topic</strong> from the
dropdown list.</p>
</li>
<li>
<p>Select the checkboxes next to the <strong>Topics</strong> or <strong>Sub-topics</strong> that
are relevant to this Module. You can click the <strong>downward arrow</strong> to
expand the topics in order to see the Sub-topics.</p>
<p><u>Note</u>: From Dec 2025 Update, teachers can toggle between viewing
displayed learning outcomes or learning outcomes. Displayed learning outcomes
are learning outcomes with more student-friendly language that are shown
to students when added by Content Approvers and Content Officers (CAs and
COs). This only applies to Content Maps with pre-populated Displayed Learning
Outcomes.</p>
<div class="isomer-image-wrapper">
<img style="width: 70%;" height="auto" width="100%" alt="" src="/images/Displayed_Learning_Outcomes__2_.png">
</div>
</li>
<li>
<p>Click the <strong>Save</strong> icon</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/Save.svg">
</div>
<p>to proceed.</p>
<p></p>
</li>
</ol>
