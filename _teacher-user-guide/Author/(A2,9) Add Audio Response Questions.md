<h1>(A2,9) Add Audio-Response Questions</h1>
<hr>
<h2>Add Audio-Response Question (ARQ)</h2>
<hr>
<ol>
<li>
<p>In the Module Editor page, hover over Question in the Component Bar.</p>
<div class="isomer-image-wrapper">
<img style="width: 50%;" height="auto" width="100%" src="/images/2Teacher/AU-AddARQ1.png">
</div>
</li>
<li>
<p>Select <strong>Audio-Response</strong>. Below are the types of ARQ options
you can choose from:</p>
<ul data-tight="true" class="tight">
<li>
<p><strong>Default</strong>: You will evaluate students’ submission and provide
Marks and/or Feedback.</p>
</li>
<li>
<p><strong><a href="/teacher-user-guide/author/speech-evaluation/" rel="noopener noreferrer nofollow" target="_blank">Speech Evaluation</a></strong>:
The system will provide Marks and/or Feedback.</p>
</li>
</ul>
</li>
<li>
<p>In the Question Body field, enter the question.</p>
</li>
<li>
<p>Click outside the Component to <strong>Save</strong>.</p>
</li>
</ol>
<h2>Edit Audio-Response Question (Enhanced)</h2>
<hr>
<p>Fill in the optional details of the Audio-response question:</p>
<ul>
<li>
<p>Recommended Time - You can set the recommended time for the student to
complete the question.</p>
</li>
<li>
<p>Add Rubrics - This allows students to view the rubrics while they attempt
the question.</p>
<p><u>Note</u>: Show and use rubric marks checkbox - If checked, marks will
be defined by the Rubrics.</p>
</li>
<li>
<p>Suggested Answer - The suggested answer to the question. This is shown
to students after they have submitted their answers.</p>
</li>
<li>
<p>Marks - You can set the number of marks. The default is one mark.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/author/edit-detail-cards/" rel="noopener noreferrer nofollow" target="_blank">here</a> for
editing the Question Component Details card.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/assess/edit-quizzes/" rel="noopener noreferrer nofollow" target="_blank">here</a> for
adding questions in a Quiz.</p>
</li>
<li>
<p>You can continue to edit Audio-Response questions even after they have
been assigned or attempted by students. This allows you to correct any
errors found in the question. The last edited date and time will be displayed
if the assignment or resource has been modified after a student’s attempt.</p>
</li>
</ul>
<p><u>Note</u>:</p>
<ul>
<li>
<p>From R4 (June 2026), the auto-save indicator will reflect 1 of the 3 statuses
when the audio recording is being processed in student account: (1) Autosaving
as draft, (2) Saved as draft, or (3) Autosaving failed. Students may also
manually click on <strong>Save as Draft</strong>.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/AddARQ1.png">
</div>
</li>
<li>
<p>If internet connection is unstable, please inform students <strong>not to close</strong> the
browser page. Their last saved response in SLS will be restored when connectivity
resumes, even if the page is refreshed.</p>
</li>
</ul>
