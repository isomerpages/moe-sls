<h1>(A1,1) Add New using Templates</h1>
<hr>
<p><strong>After the R4 June 2026 Update, My Templates will no longer be available. Teachers can continue to copy their existing modules from “My Drive”, “MOE Library” and “Community Gallery”when creating new modules.</strong>
</p>
<p>When <a href="/teacher-user-guide/author/create-new-modules/" rel="noopener noreferrer nofollow" target="_blank">creating a new Module</a> or
<a href="/teacher-user-guide/author/add-new-activities-and-sections/" rel="noopener noreferrer nofollow" target="_blank">adding a new Activity</a>, you can choose to use a <a href="/teacher-user-guide/author/about-templates/" rel="noopener noreferrer nofollow" target="_blank">templates</a> from My
Templates or SLS Templates.</p>
<h2>Viewing a Template</h2>
<hr>
<ol>
<li>
<p>On the Create/Edit Module subpage, you will see available templates under
the Create New tab.</p>
</li>
<li>
<p>Click <strong>View All</strong> to see all available templates.</p>
</li>
<li>
<p>Click&nbsp;the <strong>View Details</strong> icon</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/ViewDetails.svg">
</div>
<p>on the Module card to view the following on the Module Details subpage:</p>
<ul>
<li>
<p>Module title, cover and description (if any)</p>
</li>
<li>
<p>View Teacher Notes toggle - You can toggle on/off to view Teacher Notes
(if any)</p>
<div class="isomer-image-wrapper">
<img style="width: 50%;" height="auto" width="100%" src="/images/2Teacher/AU-AddNewTemplate2.png">
</div>
</li>
<li>
<p>All Activities/Sections in the Module - Click <strong>Show Activities</strong> to
see all Activities in a Section. The number of Activities in a Section
is shown in parenthesis ( ).</p>
</li>
<li>
<p>View in new tab - Click the <strong>Expand</strong> icon</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/external-link.svg">
</div>
<p>to view a Module/Activity/Section in a new tab.</p>
</li>
<li>
<p><u>Note</u>: This will not create a new Module in your Drive and you will
be unable to edit the Module/Activity/Section.</p>
</li>
<li>
<p>Other Information - Location, Created by, Last Edited by and Keyword Tags</p>
</li>
</ul>
</li>
</ol>
<h2>Choosing a Template</h2>
<hr>
<ol>
<li>
<p>Once you have decided on the Template(s) to use, you can tick the box
beside the Activities/Sections.</p>
<p><u>Note</u>:</p>
<ul data-tight="true" class="tight">
<li>
<p>If you select a Section, all the Activities in that Section will be added.</p>
</li>
<li>
<p>Selection has to be of the same level (if you select a Section, you can
only select another Section. If you select Activities, you can only select
other Activities. I.e. You cannot add a Section with Activities from another
Section).</p>
</li>
</ul>
</li>
<li>
<p>To clear the selected list of Activities/Sections, click <strong>Clear</strong>.</p>
</li>
<li>
<p>Click <strong>Choose</strong> to add the Activities/Sections into your Module.</p>
<p><u>Note</u>: If you want to choose the entire Module, you can also click
on the Module card on the Create/Edit Module subpage.</p>
<div class="isomer-image-wrapper">
<img style="width: 50%;" height="auto" width="100%" src="/images/2Teacher/AU-AddNewTemplate3.png">
</div>
</li>
<li>
<p>The chosen Template will inherit the Activity Details. To view and edit
the Activity Notes and Active Learning Process in the&nbsp;<strong>Activity Details</strong>&nbsp;card,
click the <strong>Settings</strong> icon&nbsp;</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/Settings24.svg">
</div>
<p>on the&nbsp;Activity Details&nbsp;card. These fields are optional and
will not be visible to students.</p>
</li>
</ol>
