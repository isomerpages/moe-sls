<h1>(A2,12) Add Display (Enhanced)</h1>
<hr>
<h2>Add Display</h2>
<hr>
<p>You can add multiple Columns and Accordions display to contain different
Components in an Activity.</p>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/AU_AddDisplay.png">
</div>
<ol data-tight="true" class="tight">
<li>
<p>In the Module Editor page, hover over <strong>Display</strong> in the Component
Bar.</p>
</li>
<li>
<p>You can add the following types of <strong>Display</strong>:</p>
<ul data-tight="true" class="tight">
<li>
<p>Page-Break</p>
</li>
<li>
<p>2-Columns</p>
</li>
<li>
<p>3-Columns</p>
</li>
<li>
<p>Accordion</p>
</li>
</ul>
</li>
</ol>
<p><u>Note</u>: You can only add Text/Media, Multiple-Choice Questions, and
Free-Response Questions components in multiple columns and accordions.</p>
<h2>Add Page-Break</h2>
<hr>
<ol>
<li>
<p>Page-Break will create a page layout and you can add different Components
from the Component Bar on each page.</p>
</li>
<li>
<p>Open the <strong>Overflow</strong> (3-dot icon) beside the Page <strong>Number</strong> to
Add Page After/Before, Move Left/Right, or Delete.</p>
<ul data-tight="true" class="tight">
<li>
<p><u>Note</u>: The question numbering will continue from the previous page.</p>
</li>
</ul>
</li>
<li>
<p>Click the <strong>Up/Down</strong> Arrows to move your Components up/down
within a page.</p>
</li>
<li>
<p>Open the <strong>Overflow</strong> (3-dot icon) beside the Component to
Move it to the Previous/Next Page or Delete.</p>
<p><u>Note</u>:</p>
<ul>
<li>
<p>When you move a Component to the Next Page, it will appear on the top
of that page.</p>
</li>
<li>
<p>When you move a Component to the Previous Page, it will appear at the
bottom of that page.</p>
<div class="isomer-image-wrapper">
<img style="width: 80%;" height="auto" width="100%" src="/images/2Teacher/AU-AddDisplay1.png">
</div>
</li>
</ul>
</li>
<li>
<p>Click outside of the Component to <strong>Save</strong>.</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" src="/images/Icons/Bulb32.svg">
</div>
<p><u>Did You Know</u>: The page numbers for quizzes and activities that
have multiple pages will remain visible at the top of the page.</p>
<div class="isomer-image-wrapper">
<img style="width: 70%;" height="auto" width="100%" src="/images/2Teacher/As_EditQuiz3.png">
</div>
</li>
</ol>
<h2>Add Components to Multiple Columns</h2>
<hr>
<ol>
<li>
<p>In the Multiple Columns view, hover over Text/Media, Question or Last
Used to add a Component.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/author/add-components/" rel="noopener noreferrer nofollow" target="_blank">here</a> to
find out more about the different Components.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/AU-AddDisplay2.png">
</div>
</li>
<li>
<p>You can click on the Component to&nbsp;<strong>Edit</strong>,&nbsp;<strong>Move Up/Down</strong>&nbsp;or&nbsp;<strong>Delete</strong>&nbsp;a
Multiple Column Component by clicking on respective icons.</p>
</li>
<li>
<p>If long content is inserted in a column, an independent scroll bar will
appear after saving the component. This will allow students to scroll through
the content side-by-side (e.g. Passage and Question).</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/AU_AddDisplay3.png">
</div>
</li>
<li>
<p>When done with editing, click outside of the Component to&nbsp;<strong>Save</strong>.</p>
</li>
</ol>
<h2>Add Components to Accordions</h2>
<hr>
<ol data-tight="true" class="tight">
<li>
<p>In the Accordion view, hover over Text/Media, Question or Last Used to
add a Component.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/author/add-components/" rel="noopener noreferrer nofollow" target="_blank">here</a> to
find out more about the different Components.</p>
</li>
<li>
<p>Click on <strong>Accordion</strong> which is the default title to rename.
Click on the <strong>Up/Down arrow</strong> to hide or unhide the Accordion.</p>
</li>
<li>
<p>You can click on the Component to <strong>Edit</strong>, <strong>Move Up/Down</strong> or <strong>Delete</strong> a
Accordion Component by clicking on respective icons.</p>
</li>
<li>
<p>You can add multiple Components by selecting from the Component Bar at
the bottom.</p>
</li>
<li>
<p>Click outside of the Component to <strong>Save</strong>.</p>
</li>
</ol>
