<h1>(A2,7) Add Error-Editing Questions</h1>
<hr>
<h2>Add Error-Editing Question</h2>
<hr>
<ol>
<li>
<p>In the Module Editor page, hover over Question in the Component Bar.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/AU_AddErrorEdit.png">
</div>
</li>
<li>
<p>Click <strong>Error-Editing</strong>.</p>
</li>
<li>
<p>In the Question Body field, enter the question.</p>
</li>
<li>
<p>There are three options under the Question Body:</p>
<ul>
<li>
<p>Hide "No Error" - If this is checked, all the texts will have to contain
errors.</p>
</li>
<li>
<p>Show identified errors for students - If this is checked, students will
not need to identify the errors for the text. If there are no errors, the
No Error box will be automatically selected for students.</p>
</li>
<li>
<p>All answers are case-sensitive - If this is checked, the answer will be
case-sensitive, that is, "usa" will not be accepted as a correct answer
while "USA" will be accepted.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" src="/images/2Teacher/AU-AddErrorEdit1.png">
</div>
</li>
</ul>
</li>
<li>
<p>Text Editing Section - This section contains the rows of texts with erroneous
words. Each row contains the following:</p>
<ul data-tight="true" class="tight">
<li>
<p>Line number</p>
</li>
<li>
<p>Text For Editing - Type out a text and click the <strong>Tick</strong> icon.
Click the error (one word, one punctuation mark, or a series of words)
and then click <strong>Add As "Error"</strong> below the text field.</p>
</li>
</ul>
</li>
<li>
<p>The error(s) that has been clicked on will be displayed under Possible
Error(s).</p>
</li>
<li>
<p>Type the Suggested Answer(s) for the errors. This is shown to students
after they have submitted their answers.</p>
</li>
<li>
<p>If there is no error in the row, check the No Error box. Students will
have to mark this row as No Error to answer it correctly.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" src="/images/2Teacher/AU-AddErrorEdit2.png">
</div>
</li>
<li>
<p>Click the <strong>Pencil</strong> icon to edit the Text For Editing.</p>
</li>
<li>
<p>Click <strong>Add Line</strong> to add subsequent lines for editing. To
reorder lines, click the <strong>Overflow</strong> (<strong>3-dot</strong> icon)
to <strong>Move Up/Move Down/Delete</strong>.</p>
</li>
<li>
<p>Click outside the Component to <strong>Save</strong>.</p>
</li>
</ol>
<h2>Edit Error-Editing Question Details</h2>
<hr>
<p>Fill in the optional details of the Fill-in-the-Blanks question:</p>
<ul>
<li>
<p>Recommended Time - You can set the recommended time for the student to
complete the question.</p>
</li>
<li>
<p>Feedback - Enter specific feedback for each line. This will be shown to
students in response to the line and error above.</p>
</li>
<li>
<p>Overall Feedback (optional) - Enter specific feedback for this question
here. Students will be able to view this after their attempt or submission.</p>
</li>
<li>
<p>Number of tries - This allows the number of tries to be set. The default
is one try.</p>
</li>
<li>
<p>Marks Per Line - You can set the number of marks per line. The default
is one mark per line.</p>
<p><u>Note</u>: Question Body, Hint, Option text, and Feedback fields support
Rich Text-enabled components.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/author/edit-detail-cards/" rel="noopener noreferrer nofollow" target="_blank">here</a> for
editing the Question Component Details card.</p>
</li>
<li>
<p>Click <a href="/teacher-user-guide/assess/edit-quizzes/" rel="noopener noreferrer nofollow" target="_blank">here</a> for
adding questions in a Quiz.</p>
</li>
<li>
<p>From Dec 2025 Update: You can continue to edit Error-Editing questions
even after they have been assigned or attempted by students. This allows
you to correct any errors found in the question. The last edited date<strong> </strong>and
time will be displayed if the assignment or resource has been modified
after a studentâ€™s attempt.</p>
</li>
</ul>
