<h1 id="edit-media-objects">(B1,ix) Edit Media Objects</h1>
<h2 id="-re-uploading-an-image-">Re-uploading an Image</h2>
<hr>
<p>As SLS upgrades, older file formats may no longer be supported. If the message "<strong>This image format will not be supported.</strong>" appears, please do the following:</p>
<ol>
<li><strong>Download</strong> the image if the option is available.</li>
<li>Alternatively, right-click to <strong>Save Image</strong> as a new file.</li>
<li>Remove the image of the older format from the Rich Text-enabled Component and reupload the image that you just downloaded.</li>
</ol>
<h2 id="-editing-a-media-object-">Editing a Media Object</h2>
<hr>
<p>Media Objects that have been uploaded from File will be editable. </p>
<ol>
	<li>Click the <b>Pencil</b> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/Pencil.svg"> to edit the image.
	</li><li>Click the <b>Trash</b> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/Trash.svg"> to delete a Media Object.
	</li><li>Click the <strong>Settings</strong> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/Settings24.svg"> to edit the image's:<ul>
<li>Title</li>
<li>Description</li>
<li>Star it as a Resource <img style="width:1.5rem; display: inline;" src="/images/Icons/AddStar.svg"></li>
<li>Set Permission for Students to download</li>
<li>Keyword Tags</li>
</ul>
		<p><u>Note</u>: Media Objects from MOE Library/Community Gallery cannot be edited by drawing or from its settings.</p>
</li>
	<p><img style="width: 100%;" src="/images/2Teacher/AU-EditMedia1.png"></p>
<li>Once done, click the <strong>Save</strong> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/Save.svg"><strong>.</strong></li>
	<li>In the Module View page, you can&nbsp;download the image by clicking the <b>Download</b> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/Download.svg">.</li>
</ol>
