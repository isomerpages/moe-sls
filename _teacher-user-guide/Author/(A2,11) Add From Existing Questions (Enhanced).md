<h1 id="add-from-existing-questions">(A2,11) Add From Existing Questions (Enhanced)</h1><hr>
<h2 id="add-from-existing-questions-from-library-starred-resources">Add From Existing Questions from Library/Starred Resources</h2>
<hr>
<ol>
<li>In the Lesson Editor page, hover over <strong>Question</strong> in the Component Bar.</li>
<li>Under Add From Existing, select <strong>Question from Library</strong> to add from MOE Library, Community Gallery, and My Drive or <strong>Question from Starred Resources</strong> to add from Starred Resources.</li>
<p><img alt="Add From Existing Questions" style="width: 50%;" src="/images/2Teacher/AU_AddFromExisiting1.png"></p>
</ol>
<h2 id="view-an-existing-question-from-library-starred-resources">View an Existing Question From Library/Starred Resources</h2>
<hr>
<ol>
<li>Click the <strong>View Details</strong> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/ViewDetails.svg"> on the Question card to view the following on the Question Details subpage:
<ul>
<li>Question and answer (if available)</li>
</ul>
<p><u>Note</u>: To edit a Question, you have to add it to a module first.</p>
<ul>
<li>Other Information - Location, Question Type, Question Tags, Created by, Last Edited by, Keyword Tags and Used in.</li>
</ul>
<p><img alt="Add From Existing Questions" style="width: 50%;" src="/images/2Teacher/AU_AddFromExisting2.png"></p>
</li>
<li>When Adding from Library, you can search for Questions using the Search Bar which contains the following filter:
<ul>
<li>Subject</li>
<li>Level</li>
</ul>
<p><u>Note</u>: Click <strong>Guided Search</strong> to search for specific words/phrases.</p>
</li>
<li>Click <strong>Apply</strong> to apply your filters.</li>
<li>You may use filters for more targeted search results. To apply filters, click <strong>Filter</strong>. You can apply the following filters:
<ul>
<li>Location</li>
<li>Resource Type</li>
<li>Subject</li>
<li>Level</li>
<li><a target="_blank" href="/teacher-user-guide/discover/search-for-resources/">Topic Filters</a> to select Content Map, and Topics and Sub-topics</li>
</ul>
</li>
<li>Click <strong>Apply Filters</strong> or <strong>Reset</strong>.</li>
<li>By default, the Search Results are sorted according to Relevance. You can choose to sort the results by recency by selecting Latest from the Sort By dropdown list.</li>
<p><img alt="Add From Existing Questions" style="width: 50%;" src="/images/2Teacher/AU_AddFromExisiting3.png"></p>
<li>For Multi-part Questions, you can click the <strong>View Details</strong> icon <img style="width:1.5rem; display: inline;" src="/images/Icons/ViewDetails.svg"> to view each sub question on the left.</li>
</ol>
<h2 id="choose-an-existing-question-from-library-starred-resource">Choose an Existing Question From Library/Starred Resource</h2>
<hr>
<ol>
<li>Search for questions by entering keywords from their titles. You can then refine your search results using the "Resource Type" filter.</li>
<li>Once you have decided on the question(s) to add to your module, tick the box(s) on the Question card(s) then click <strong>Select</strong>.</li>
<p><img alt="Add From Existing Questions" style="width: 50%;" src="/images/2Teacher/AU_AddFromExisiting4.png"></p>
<li>You may also click <strong>Choose</strong> in the Questions Detail subpage.</li>
<li>For Multi-part Questions, once you have decided on the question part(s) to add, tick the box beside the question in the Question Details subpage then click <strong>Choose</strong>.</li>
<p><u>Note</u>: Selected questions will be added to the same level. (If you add parts a, bi and cii of the same multi-part question, the three questions will be added at the same level e.g. 1, 2, 3 or a, b, c)</p>
<p><img alt="Add From Existing Questions" style="width: 50%;" src="/images/2Teacher/AU_AddFromExisiting5.png"></p>
<li>To clear the selected list of question, click <strong>Clear</strong>.</li>
</ol>
