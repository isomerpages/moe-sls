<h1>Plan and Generate Modules with Authoring Copilot (Enhanced)</h1>
<p><u>Important</u>:</p>
<ul data-tight="true" class="tight">
<li>
<p>As Authoring Copilot uses generative AI, its response is probabilistic
in nature. Thus, it may at times generate a different number of Sections,
Activities and Components than expected.</p>
</li>
<li>
<p>Check that the content generated by Authoring Copilot is accurate and
appropriate.</p>
</li>
</ul>
<p></p>
<p>Authoring Copilot is a teacher-facing tool for lesson planning, including
generating module plans, sections, activities, and components.</p>
<p>View the video below for a step-by-step guide on planning and generating
modules with Authoring Copilot.</p>
<p></p>
<div class="iframe-wrapper">
<iframe referrerpolicy="strict-origin-when-cross-origin" height="315" width="560" allowfullscreen="true" frameborder="0" src="https://www.youtube.com/embed/ZGUGNe4zslA?si=K-y7WnkShVJTf9Ss"></iframe>
</div>
<p></p>
<ol>
<li>
<p>On the homepage, click <strong>Create New/Edit</strong>. Click <strong>Module</strong>.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" alt="Use Authoring Copilot to create New Sections" src="/images/2Teacher/AU_ACP1.png">
</div>
</li>
<li>
<p>You can also access Authoring Copilot in the Module Editor page of the
Section, and click the <strong>orange Create</strong> icon</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" alt="Authoring Copilot" src="/images/Icons/ACP.svg">
</div>
<p>on the Module card to open Authoring Copilot.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" alt="Use Authoring Copilot to create New Sections" src="/images/2Teacher/AU_ACP13.png">
</div>
</li>
<li>
<p>Add the Module Title.</p>
<div class="isomer-image-wrapper">
<img style="width: 80%;" height="auto" width="100%" alt="Use Authoring Copilot to create New Sections" src="/images/2Teacher/AU_ACP3.png">
</div>
</li>
<li>
<p>In the Module Notes field, add a brief description of your module's learning
objectives and key concepts. Click <a href="https://www.opal2.moe.edu.sg/csl/s/epedagogy/wiki/page/view?title=%5BSection+1.3%5D+e-Pedagogy+in+subject+discipline" rel="noopener noreferrer nofollow" target="_blank">here</a> (requires
OPAL login) for more information on how to craft more specific input, for
more supporting resources and ideas.</p>
</li>
<li>
<p><u>Note</u>: A maximum of 2000 characters can be input into this field.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" alt="Use Authoring Copilot to create New Sections" src="/images/2Teacher/AU_ACP4.png">
</div>
</li>
<li>
<p>Set the number of sections to generate in the module. The number of sections
can depend on the number of topics or sub-topics covered in the module,
or the number of lessons you have to cover the module content.</p>
</li>
<li>
<p><u>Note</u>: A maximum of 20 Sections can be created in a module.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" alt="Use Authoring Copilot to create New Sections" src="/images/2Teacher/AU_ACP5.png">
</div>
</li>
<li>
<p>Click the dropdown arrow located on the right of the Module Tag field
to select the subject, level and content map. Select the checkboxes next
to topics and subtopics and learning outcomes relevant to the Module.</p>
</li>
<li>
<p><u>Note:</u>
</p>
<ul data-tight="true" class="tight">
<li>
<p>Module Tags help to refine the lesson content generated by Authoring Copilot
to the appropriate subject, level, topics, subtopics and learning outcomes.</p>
</li>
<li>
<p>These tags also improve searchability in MOE Library, Community Gallery,
and My Drive.</p>
</li>
<li>
<p>If Modules are submitted to the Community Gallery, tags will allow teachers
to find the Module using filters and relevant search terms.</p>
</li>
</ul>
<p>(Optional) Click the dropdown arrow located on the right of the content
map and topic to select content map and relevant topics.</p>
</li>
<li>
<p>Add <strong>Keyword Tags</strong>.</p>
</li>
<li>
<p>In the <strong>Additional Details</strong> field, you can provide specific
guidance for structuring your module, such as:</p>
<ul data-tight="true" class="tight">
<li>
<p>Descriptions of key learning outcomes</p>
</li>
<li>
<p>Desired 21st-century competencies (21CC) or Digital Literacy and Technological
Skills</p>
</li>
<li>
<p>Instructional strategies or activities</p>
</li>
<li>
<p>Success criteria linked to the learning objectives</p>
</li>
</ul>
</li>
<li>
<p>If no details are provided, Authoring Copilot will generate a complete
module based on the curriculum. For more tailored results, you can start
broad and add specifics when regenerating.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%;" height="auto" width="100%" alt="Use Authoring Copilot to create New Sections" src="/images/2Teacher/AU_ACP11.png">
</div>
</li>
<li>
<p>Click <strong>Add Existing</strong> and select <strong>From Templates</strong> to
choose a suitable SLS template for Authoring Copilot to refer to when you
wish for it to generate a similar Module Structure.</p>
</li>
<li>
<p>Click <strong>Create</strong> to generate the Section(s) in the Module.
Authoring Copilot will create a Suggested Module Plan with Module description
and Section notes. Click the checkboxes next to the generated content you
wish to include in the module. Click <strong>Add</strong> to autogenerate
Section(s) within the Module.</p>
</li>
<li>
<p>Authoring Copilot will create a Suggested Module Plan with Module description
and Section notes. Click <strong>Add</strong> to autogenerate Section(s)
within the Module.</p>
</li>
<li>
<p>You can edit the Module and Section(s) generated in the Module Editor
page, as you would for a normal lesson.</p>
</li>
<li>
<p>To regenerate the Module and Section(s), click <strong>Back to Generate Module</strong> to
edit the Module Details and regenerate the Module and Section(s).</p>
</li>
<li>
<p>Alternatively, you can regenerate the Module and Section by clicking the <strong>orange Create</strong> icon</p>
<div class="isomer-image-wrapper">
<img style="width:1.5rem; display: inline;" height="auto" width="100%" alt="Create" src="/images/Icons/ACP.svg">
</div>
<p>on the respective cards to reopen Authoring Copilot. Newly generated Sections
will be appended to the end of the existing set of Sections.</p>
<p>When creating your module, include curricular considerations from subject-specific
<a href="https://www.opal2.moe.edu.sg/csl/s/epedagogy/wiki/page/view?title=%5BSection+1.3%5D+e-Pedagogy+in+subject+discipline" rel="noopener noreferrer nofollow" target="_blank">Teaching and Learning Guides</a>(requires OPAL login).</p>
<p>These may include:</p>
<ul data-tight="true" class="tight">
<li>
<p>Building on students' prior knowledge</p>
</li>
<li>
<p>Extending key concepts</p>
</li>
<li>
<p>Describing skills students should learn</p>
</li>
<li>
<p>Providing real-world applications</p>
</li>
</ul>
<p>You can add these considerations in the <strong>Module Notes</strong> or <strong>Additional Details</strong> fields
to guide Authoring Copilot in creating more relevant content.</p>
<p>You may also refer to the <a href="https://www.opal2.moe.edu.sg/csl/s/epedagogy/wiki/page/view?title=%5BSection+1.3%5D+e-Pedagogy+in+subject+discipline" rel="noopener noreferrer nofollow" target="_blank">Teaching and Learning Guides</a> (requires
OPAL login) for your subject in OPAL2.0 for more information.</p>
<p><u>Note:</u>
</p>
<ul data-tight="true" class="tight">
<li>
<p>Any text formatting or images in the Module Notes will not be used to
generate the Lesson Plan.</p>
</li>
<li>
<p>A maximum of 2000 characters can be input into this field.</p>
</li>
