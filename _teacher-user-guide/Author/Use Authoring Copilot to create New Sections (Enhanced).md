<h1>Plan and Generate Modules with Authoring Copilot (Enhanced)</h1>
<p><u>Important</u>:</p>
<ul data-tight="true" class="tight">
<li>
<p>As Authoring Copilot uses generative AI, its response is probabilistic
in nature. Thus, it may at times generate a different number of Sections,
Activities and Components than expected.</p>
</li>
<li>
<p>Check that the content generated by Authoring Copilot is accurate and
appropriate.</p>
</li>
</ul>
<p></p>
<p>Authoring Copilot is a teacher-facing tool for lesson planning, including
generating module plans, sections, activities, and components.</p>
<p>Authoring Copilot can also generate other various resources in a module
such as:</p>
<ul data-tight="true" class="tight">
<li>
<p>game elements (Refer to <a href="https://www.learning.moe.edu.sg/gamify-a-module-with-authoring-copilot/" rel="noopener nofollow" target="_blank">Gamify a Module with Authoring Copilot)</a>
</p>
</li>
<li>
<p>images (Refer to <a href="https://www.learning.moe.edu.sg/generate-images-with-authoring-copilot/" rel="noopener nofollow" target="_blank">Generate Images with Authoring Copilot</a>)</p>
</li>
<li>
<p>interactives (Refer to <a href="https://www.learning.moe.edu.sg/generate-interactives-with-authoring-copilot/" rel="noopener nofollow" target="_blank">Generate Interactives with Authoring Copilot)</a>
</p>
</li>
<li>
<p>rubrics (Refer to <a href="https://www.learning.moe.edu.sg/teacher-user-guide/author/add-rubrics-to-audio-and-free-response-questions/" rel="noopener nofollow" target="_blank">Add Rubrics to Audio- and Free-Response Questions (Enhanced)</a>)</p>
</li>
</ul>
<p>View the video below for a step-by-step guide on planning and generating
modules with Authoring Copilot.</p>
<p></p>

<p></p>
<p>1.&nbsp;On the homepage, click Create New/Edit. Click&nbsp;on the Authoring
Copilot icon on the Module card to open Authoring Copilot.</p>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/ACP_2026_1.png">
</div>
<p></p>
<ol start="2" data-tight="true" class="tight">
<li>
<p>Choose from the predefined recipes (e.g. Active Learning Framework, Inquiry-based
Learning) or create a custom recipe by providing your own instructions.
If selecting a predefined recipe, modify the instructions in the [placeholder
values] to suit your class’s needs.</p>
</li>
</ol>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/ACP_2026_2.png">
</div>
<p>Note that the appropriate learning experience is pre-filled with the selected
recipe (e.g. <em>Elicit prior knowledge and misconceptions</em>). To further
customise your lesson, include success criteria or Emerging 21st Century
Competencies (e.g. CAIT 1 – <em>Students should explain his/her reasoning and decision</em>).
You may further refer to subject-specific <a href="https://www.opal2.moe.edu.sg/csl/s/epedagogy/wiki/page/view?title=%5BSection+1.3%5D+e-Pedagogy+in+subject+discipline" rel="noopener noreferrer nofollow" target="_blank">Teaching and Learning Guides</a> (requires
OPAL login), to include curricular considerations.</p>
<ol start="3" data-tight="true" class="tight">
<li>
<p>Click the dropdown arrow located to the right of the Module Tag field
to select the subject, level and content map. Select the checkboxes next
to topics and subtopics and learning outcomes relevant to the Module.</p>
</li>
</ol>
<p>Note:</p>
<ul data-tight="true" class="tight">
<li>
<p>Module Tags help to refine the lesson content generated by Authoring Copilot
to the appropriate subject, level, topics, subtopics and learning outcomes.</p>
</li>
<li>
<p>These tags improve searchability in MOE Library, Community Gallery and
My Drive.</p>
</li>
</ul>
<p>Click the dropdown arrow&nbsp;located on the right of the content map
and topic to select content map and relevant topics.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/ACP_2026_3.png">
</div>
<ol start="4" data-tight="true" class="tight">
<li>
<p>Click the dropdown arrow&nbsp;located&nbsp;to the right of the Module
Tag field to select&nbsp;two&nbsp;Key Applications of Technology (KAT).</p>
</li>
</ol>
<p>Note:</p>
<ul data-tight="true" class="tight">
<li>
<p>Selected&nbsp;KAT&nbsp;refine the lesson content generated by Authoring
Copilot to&nbsp;ensure that&nbsp;technology-mediated&nbsp;learning experiences&nbsp;are&nbsp;well-designed
in line with e-Pedagogy&nbsp;guidelines. For more information, read the
Guide to e-Pedagogy&nbsp;<a href="https://google.com/url?q=https%3A%2F%2Fsites.google.com%2Fmoe.edu.sg%2Fguidetoe-pedagogy%2Fe-pedagogy_1&amp;sa=D&amp;sntz=1&amp;usg=AOvVaw0V8pOw5PTLeJZFiSAFfruw" rel="noopener noreferrer nofollow" target="_blank">here</a>.</p>
</li>
</ul>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/ACP_2026_4.png">
</div>
<p>5.&nbsp;To provide context and scope the Authoring Copilot’s responses
more accurately, click <strong>Add Knowledge Base</strong> to upload reference
materials.&nbsp;You can upload up to 20 Knowledge Bases that can&nbsp;comprise&nbsp;text
(up to 100,000 characters) and files.&nbsp;To add PDFs or images from your
personal device, click <strong>From File Upload.</strong> To add a Google
File, click <strong>From Google Drive</strong>. To add resources already
in SLS, select either <strong>From Library</strong> or <strong>From Starred Resources</strong>.</p>
<p>Note:</p>
<ul data-tight="true" class="tight">
<li>
<p>Users can send a maximum of 20 Knowledge Bases (KBs), which includes both
text and file KBs.&nbsp;A&nbsp;maximum of 20 files of no more than 20 MB
in total can be sent.</p>
</li>
<li>
<p>Adding a Knowledge Base will anchor the generation of content to information
added, e.g. a syllabus document, a comprehension&nbsp;passage&nbsp;or an
excerpt from MOE’s interactive digital textbooks. Use specific PDF segments
to enhance results of generated content.</p>
</li>
<li>
<p>Please ensure all uploaded materials&nbsp;comply with&nbsp;copyright&nbsp;comply
with&nbsp;data security classification (up to Official (Closed), Non-Sensitive)
and copyright laws, using only legally permitted content, properly licensed
resources, or materials with explicit permissions. You may refer to 
<a href="https://www.learning.moe.edu.sg/terms-of-use/" rel="noopener noreferrer nofollow" target="_blank">https://www.learning.moe.edu.sg/terms-of-use/</a> for further details</p>
</li>
</ul>
<p>6.&nbsp;Under Advanced Settings, Teachers can choose to add a Module Title
(e.g.&nbsp;taking reference from Scheme of Work for a topic)&nbsp;and select
the desired number of sections and activities for the module. If not populated,
Authoring Copilot will suggest accordingly in the generated output based
on earlier fields.</p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/ACP_2026_6.png">
</div>
<ol start="7" data-tight="true" class="tight">
<li>
<p>Click Create&nbsp;to generate the&nbsp;content. Authoring Copilot will
create a Suggested Module Plan with sections&nbsp;and&nbsp;activities generated.
Click the checkboxes next to the generated content you wish to include
in the module.</p>
</li>
</ol>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/ACP_2026_7.png">
</div>
<ol start="8" data-tight="true" class="tight">
<li>
<p>You can edit the Module generated in the Module Editor page, as you would
for a typical lesson.</p>
</li>
<li>
<p>Click on Generate Components next, to create the rest of the module. Review
the content generated to ensure that it is designed in line with learning
objectives, meets your class profile and use of technology is fit-for-purpose.</p>
<p></p>
<div class="isomer-image-wrapper">
<img style="width: 100%" height="auto" width="100%" alt="" src="/images/2Teacher/ACP_2026_8.png">
</div>
</li>
</ol>
