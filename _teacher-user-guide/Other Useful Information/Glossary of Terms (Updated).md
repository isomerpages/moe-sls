<h1>Glossary of Terms (Updated)</h1>
<table>
  <thead>
    <tr>
			<th><strong>Component</strong></th>
      <th><strong>Description</strong></th>
    </tr>
  </thead>
  <tbody>
        <tr>
      <td>
        <p>Text/Media</p>
      </td>
      <td>
        <p>Text/Media allows teachers to text, media objects, interactives, embedded websites, and more into the text field. It can be used to display content information or instructions.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Poll</p>
      </td>
      <td>
        <p>Polls allow teachers to set questions with no correct answers. It can be used to collect sentiments, quickly ask a question and instantly receive responses from students during class.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Interactive Thinking Tool</p>
      </td>
      <td>
        <p>Interactive Thinking Tool allows teachers to set questions or thinking routines. It can be used for classroom discussions or for students to show their thinking process. After responses are collected, students can view and comment on the responses of their peers on an Interaction Board.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Multi-Part Question</p>
      </td>
      <td>
        <p>Multi-Part Questions allow teachers to set questions with multiple parts. For example, part a, b, c. Teachers can also put Multi-Part Questions within another Multi-Part Question. For example, ai, aii, aiii.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Multiple-Choice Question</p>
      </td>
      <td>
        <p>Multiple-Choice Questions allow teachers to set MCQ with one correct answer or multiple correct answers. Feedback can be provided for the questions and for each option.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Click-and-Drop Question</p>
      </td>
      <td>
        <p>Click-and-Drop Questions allow teachers to create Click-and-Drop blanks on a base image. Students will be able to drop the correct answers onto the base image. Feedback can be provided for the question.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Error-Editing Question</p>
      </td>
      <td>
        <p>Error-Editing Questions allow teachers to set sentences with intentional errors. Students will be able to highlight an erroneous part of the sentence to edit.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Fill-in-the-Blanks Question</p>
      </td>
      <td>
        <p>Fill-in-the-Blanks Questions allow teachers to insert text with blank spaces. Students will provide the answers to the question in the blanks provided.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Audio-Response Question</p>
      </td>
      <td>
        <p>Audio-Response Questions allow teachers to set questions that require an audio response from students. Students can submit an audio recording directly using SLS.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Free-Response Question</p>
      </td>
      <td>
        <p>Free-Response Questions allow teachers to set questions that require students to compose their responses freely. Teachers can even allow File Submission for students to submit their answers in various file formats.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Text</p>
      </td>
      <td>
        <p>Rich Text can be added to all components. The colour, size, style of the font can be edited to make the lesson look more interesting.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Image</p>
      </td>
      <td>
        <p>Image can be added to all components. Drawing can also be added on top of a base image.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Video</p>
      </td>
      <td>
        <p>Video can be added to all components. It can be added as a file or a link from Youtube.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Interactive</p>
      </td>
      <td>
        <p>Interactives can be added to all components. Interactives generally come in a format of a simulation or applet for users to interact with.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Table</p>
      </td>
      <td>
        <p>Tables can be added to all components. It can be used to structure and contain data.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Embed</p>
      </td>
      <td>
        <p>Embed allows whitelisted websites and apps to be added to all components. Some websites that can be embedded include Padlet, mentimeter, etc.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Thinking Routine</p>
      </td>
      <td>
        <p>Thinking Routine can only be added to the Interactive Thinking Tool. It can be found as one of the few Interactive Thinking Tool Templates when creating an Interactive Thinking Tool.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Pre-populated Answers</p>
      </td>
      <td>
        <p>Pre-populated Answers can be added to Free-Response Questions and the Interactive Thinking Tool. They allow teachers to add in sentence starters or a template for students to populate their answers.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Feedback</p>
      </td>
      <td>
        <p>Feedback can be added to most question components. It provides general feedback to students.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Option Feedback</p>
      </td>
      <td>
        <p>Option feedback can be added to some question components. It provides more targeted feedback to students based on the choices they have chosen.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Hint</p>
      </td>
      <td>
        <p>Hints can be added to some question components. Teachers can set hints to scaffold the questions for students.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Rubric</p>
      </td>
      <td>
        <p>Rubrics can be added to Free-Response Questions and Audio-Response Questions. It allows teachers to award marks to students based on a rubric. It also allows students to refer to the rubric before answering their questions.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>File Submission</p>
      </td>
      <td>
        <p>File Submission allows students to submit a file as their answer.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Tooltip</p>
      </td>
      <td>
        <p>Tooltips can be added to texts to provide additional descriptions, explanations, or thinking prompts.</p>
      </td>
    </tr>
		<tr>
      <td>
        <p>Student Tooltip</p>
      </td>
      <td>
        <p>Student Tooltip can be added to any component and is visible to students when they attempt the lesson. When clicked, students will be able to create a note on the right drawer. This allows teachers to point out areas in the lesson where students are encouraged to create a note.</p>
      </td>
    </tr>
    <tr>
      <td>
        <p>Teacher Tooltip</p>
      </td>
      <td>
        <p>Teacher Tooltip can be added to any component and is hidden from students. It allows teachers to explain and note the rationale of their lesson. This can be very useful when lessons are shared with other teachers.</p>
      </td>
    </tr>
		   <tr>
      <td>
        <p>Text-to-Speech (Chinese, Malay and Tamil Languages Only)</p>
      </td>
      <td>
        <p>Text-to-Speech can be added to any component. It serves as a scaffold for students by generating an audio clip based on the text provided.</p>
      </td>
    </tr>
		   <tr>
      <td>
        <p>E-Dictionary (Chinese Language Only)</p>
      </td>
      <td>
        <p>E-Dictionary allows student to quickly look up the meaning of vocabularies found within Assignments. </p>
      </td>
    </tr>
		   <tr>
      <td>
        <p>Speech Evaluation (Chinese Language Only)</p>
      </td>
      <td>
        <p>Speech Evaluation can be added to Audio-Response Question components. This engine allows students to submit their audio recordings and automatically grade studentsâ€™ submission based on rubrics.</p>
      </td>
    </tr>
		   <tr>
      <td>
        <p>Language Localisation (All Languages)</p>
      </td>
      <td>
        <p>Language Localisation will translate English instructional text in SLS to the respective Mother Tongue Languages. This is not applicable to the text that is entered by users.</p>
      </td>
    </tr>
  </tbody>
</table>
